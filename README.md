# ⚙️ Chat App - Backend

A robust backend for the Chat App built with **Node.js**, **Express.js**, **MongoDB**, and **Socket.IO**. It provides secure authentication, real-time communication, and RESTful APIs to power the chat application.

---

## 🚀 Features

* 🔐 JWT Authentication
* 👤 User Registration & Login
* 💬 Real-Time Messaging with Socket.IO
* 🛡️ Protected API Routes
* 🍪 Secure Cookie-Based Authentication
* 📡 RESTful API Architecture
* 🗄️ MongoDB Database Integration
* ⚡ Fast & Scalable Express Server

---

## 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* Socket.IO
* JWT (JSON Web Token)
* bcrypt
* Cookie Parser
* CORS
* dotenv

---

## 📦 Installation

```bash
git clone https://github.com/Gayatri-Vishwa/ChatApp_Backend.git
cd ChatApp_Backend
npm install
npm start
```

> If your project uses **nodemon**, run:

```bash
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in the project root and add:

```env
PORT=8000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_jwt_secret

CLIENT_URL=https://chat-app-frontend-dsux.vercel.app
```
